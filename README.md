# Skills

Personal skill collection for reusable Codex workflows.

## Included Skills

- `commit-writing`
  Write Conventional Commits–conformant git commit messages. Keeps the subject tight, only adds a body when the diff alone can't explain intent, and guides splitting working trees into reviewable / revertable commits.

- `pnpm-monorepo-trusted-publishing`
  Set up and debug pnpm workspace monorepo releases to npm with GitHub Actions, Trusted Publisher, and provenance.

- `pr-writing`
  Draft or rewrite GitHub PR descriptions. Defaults to an AS IS / TO BE structure so reviewers can quickly tell what is broken today and what this PR changes, with variants for features, refactors, dependency bumps, and rollbacks.

- `readme-writing`
  Write or rewrite a project README that converts a 10-second visitor into a user. Front-loads identity → value → minimum-runnable example; cuts marketing fluff, premature API dumps, and decorative TOCs.
